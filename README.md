* Connect to your Bluetooth media device (eg. [especiallyAlfa](https://github.com/snsdosen/especiallyAlfa)) and see it's information.
* Update to the latest firmware Over-The-Air (OTA) via Bluetooth in simple click of a button.

## How to install:
* Download the latest .apk from [releases](https://github.com/snsdosen/MDCompanion/releases) and run it on your Android phone.

## How to update your media device using the application:
1. Connect to your device via bluetooth and run the application.
2. Switch to AM/FM tuner input on your head unit. (Don't use CD input during update).
3. Click "check for updates" and if there are any available updates application will prompt you if you want to update your device.
4. Update will start and it will show you the progress (it takes about 5 min to update).
5. After update turn ignition off for a couple of minutes.
6. Your device will now have the latest firmware available.

## What if the update process fails or the power was interrupted?
  Media devices are using dual boot firmware so there is no risk of bricking if the update failed.<br>
  Turn off your ignition for a couple of minutes and try updating again.
